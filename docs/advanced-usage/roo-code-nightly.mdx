# Roo Code Nightly

对于希望获得最新功能和更新的用户，Roo Code 提供了一个名为 **Roo Code Nightly** 的预发布构建版本，该版本会在我们的主开发分支合并更改时自动发布。这意味着它始终反映最新的代码。它专为希望访问最新功能并参与早期测试的用户而设计。

---

## 了解 Nightly 与其他预发布版本的区别

重要的是要区分 Roo Code Nightly 和其他类型的预发布软件（如 beta 版本或候选发布版）：

*   **Roo Code Nightly：**
    *   **频率：** 自动化构建，在每次合并到主开发分支时生成。
    *   **内容：** 反映最新的合并代码。
    *   **稳定性：** 高度实验性；可能包含错误或未完成的功能。
    *   **测试：** 发布前进行最少或无手动测试。
    *   **受众：** 主要面向开发人员、内部测试人员或对可能不稳定的软件感到满意的用户，他们希望看到最新的更改并帮助及早发现问题。

*   **通用预发布（例如 Beta、候选发布版）：**
    *   **频率：** 定期发布（例如，每周、每月）以达到特定里程碑。
    *   **内容：** 代表更精心策划和计划的中间版本。
    *   **稳定性：** 比 Nightly 构建版本更稳定；通常会进行一些手动测试或验证。
    *   **测试：** 通常会接受更集中的测试。
    *   **受众：** 面向更广泛的早期采用者或 beta 测试人员，旨在在稳定发布前收集反馈。

简而言之：**Nightly = 最新代码，可能不稳定，频繁更新。** 其他预发布版本通常更新频率较低，经过更多验证，并且旨在进行更广泛的测试。

---

## 安装 Roo Code Nightly

要安装 Roo Code Nightly：

1.  打开 VS Code。
2.  访问扩展：单击活动栏中的扩展图标，或按 `Ctrl+Shift+X` (Windows/Linux) 或 `Cmd+Shift+X` (macOS)。
3.  搜索 "Roo Code Nightly"。
4.  选择 Roo Code 提供的 "Roo Code Nightly" 并单击 **安装**。
5.  如果提示，请重新加载 VS Code。

你可以同时安装 Roo Code 的稳定版本和 Roo Code Nightly。
